# SOC Analyst Lab Projects

This repository contains hands-on cybersecurity projects focused on SIEM deployment, log analysis, and MITRE ATT&CK-based threat detection.

## Project 01: Splunk Installation and Universal Forwarder Setup

In this project, Splunk Enterprise, Sysmon, and Splunk Universal Forwarder were installed and configured to collect Windows logs for security monitoring.

### Skills Learned

* Splunk Enterprise Installation
* Universal Forwarder Configuration
* Sysmon Log Collection
* Log Analysis

## Project 02: MITRE ATT&CK Analysis with Atomic Red Team

In this project, Atomic Red Team was used to simulate attacker techniques and analyze the generated logs in Splunk.

### Skills Learned

* MITRE ATT&CK Framework
* Atomic Red Team
* Threat Detection
* Log Investigation

## Tools Used

* Splunk Enterprise
* Splunk Universal Forwarder
* Sysmon
* Atomic Red Team
* Windows 10

These projects helped me gain practical experience in SOC Analyst and SIEM operations.
